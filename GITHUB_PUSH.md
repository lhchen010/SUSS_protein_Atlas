# Pushing this to GitHub

This repository is ready to publish. On your own machine (which has your GitHub credentials):

## 1. Unpack

```bash
tar xzf suss-atlas-repo.tar.gz
cd suss-atlas
```

## 2. Initialise git and make the first commit

```bash
git init
git add .
git status                       # sanity-check what will be committed
git commit -m "Initial release: SUSS Atlas pipeline + portal (v1.0.0)"
```

The `.gitignore` already excludes run outputs (`results/`, `.snakemake/`), user inputs
(`input/pdb/`), external databases, and OS junk. The bundled 100-protein example under
`examples/` **is** committed so others can self-check their install.

## 3. Create the repository on GitHub

Create an empty repo on github.com (no README/License — this repo already has them). Suggested:

- **Name:** `suss-atlas`
- **Description:** *Structural-similarity (SUSS) atlas pipeline + web portal for secreted-protein
  effector repertoires — Foldseek clustering, independent US-align TM cross-check, interactive HTML.*
- **Visibility:** public (or private if you prefer)

## 4. Connect and push

Replace `<YOUR-ORG-OR-USERNAME>`:

```bash
git remote add origin https://github.com/<YOUR-ORG-OR-USERNAME>/suss-atlas.git
git branch -M main
git push -u origin main
```

(If you use SSH: `git remote add origin git@github.com:<YOUR-ORG-OR-USERNAME>/suss-atlas.git`.)

## 5. Tag the release

```bash
git tag -a v1.0.0 -m "SUSS Atlas v1.0.0"
git push origin v1.0.0
```

## Size note

The repo is small (~a few MB) — the 100 example PDBs are the bulk. No large databases or run
outputs are included (those are downloaded/generated by the user). If GitHub warns about the
example `.pdb` files you can optionally move them to a release asset, but a few MB is fine for a
normal repo.

## After publishing

- Check the rendered `README.md` on GitHub — the `docs/engine_rulegraph.svg` DAG should display.
- Consider enabling GitHub Issues so other labs can report installation problems.
- If you later want a citable DOI, connect the repo to Zenodo and cut a release.
